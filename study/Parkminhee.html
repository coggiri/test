white-space

모든 브라우저 지원(ie는 8부터 지원)

normal
일반적인 것, default
inherit
말그대로 부모요소로 부터 상속 받는 것(ie8부터)
nowrap
줄바꿈을 하지 않는다
pre
pre태그와 같은 기능, 마크업에서 앤터로 줄바꿈 한것 만 적용이 되고 넘어가는 것에는 줄바꿈 하지 않는다. 공백은 브라우저에 의해 유지된다.
pre-line
마크업에서 줄바꿈 한 것이 적용 되면서 width를 넘어가는 텍스트도 자동으로 줄바꿈이 된다
pre-wrap
pre-line 과 비슷하다. 공백은 브라우저에 의해 유지된다.
initial
속성값을 default 값으로 한다.

word-break

word-break는 말 그대로 단어를 자르는것. 기본적으로 줄이 넘어갈 때는 단어 단위로 잘려서 넘어가게 되어있다.(default) 띄어 쓰기를 하지 않고 글을 쓴다면 한 단어로 인식해서 width를 넘어가도 줄바꿈이 되지 않는다. 그럴 때 word-break를 사용해주면 줄바꿈이 가능해진다.

모든 브라우저 지원(ie는 8부터 지원)

normal
일반적인 것, default
normal이 기본값, 단어 단위로 끊어서 줄바꿈 한다.(단어 단위로 줄바꿈되어 오른쪽 공간이 꽉차지 않고 비어있는 형태다.)
inherit
말그대로 부모요소로 부터 상속 받는 것(ie8부터)
break-all
글자 단위로 줄바꿈 한다.(글자단위로 줄바꿈이 되어 글자가 박스 안에서 가득차는 형태다.)
keep-all
한글일 경우 띄어쓰기 기준으로 줄바꿈 해준다. (ie계열/아시아 언어만 지원,파폭 3.5이상에서는 지원)
initial
속성값을 default 값으로 한다.
Original(word-break:normal)

All the times that you rain on my parade And all the clubs you get in using my name You think you broke my heart Ohhh girl for goodness sake You think I'm crying Oh my ohhh, well I ain't!

word-wrap

word-wrap속성은 긴 텍스트를 강제로 끊어 줄바꿈 해주는 속성이다.(overflow속성과 비슷함) 끊어질 수 없는 주소 같은 문자열을 끊을 때 사용. word-wrap속성은 비아시아 언어의 줄바꿈 제어/아시아언어(중국어,일본어,한국어)의 줄바꿈은 word-break 속성을 이용

모든 브라우저 지원(ie는 8부터 지원)

normal
일반적인 것, default
글자가 길어도 끊어지지 않고 한줄에 계속 표시
inherit
말그대로 부모요소로 부터 상속 받는 것(ie8부터)
initial
속성값을 default 값으로 한다.
break-word
강제로 끊어서 줄바꿈함
Original(word-wrap: normal)

word-wrap:break-word;
